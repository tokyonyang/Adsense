# v1.17 아침 헤드라인 뉴스 품질 개선

## 문제

기존 헤드라인 뉴스 텍스트가 아래처럼 작성되다 만 것처럼 보였습니다.

```text
오늘의 주요뉴스…
1부 오늘의 주요뉴스
베네수엘라 강진⋯김건희 ... 外
송혜교 UAA 결별·대성 열애설...
```

## 원인

1. 검색어가 `오늘 주요 뉴스` 중심이라 언론사 종합기사/브리핑 제목이 많이 유입됨
2. 원문 기사 제목 자체가 이미 `…`, `外` 등으로 잘린 경우가 있음
3. 이전 버전의 제목 축약 로직이 텍스트 메시지에도 영향을 줄 수 있음

## 수정

- `오늘의 주요뉴스`, `1부 오늘의 주요뉴스`, `外`, `뉴스브리핑` 같은 종합기사 제목 제외
- 검색어를 더 구체적인 카테고리/이슈형 쿼리로 변경
- 텍스트 메시지에서는 제목을 강제로 짧게 자르지 않음
- 매우 긴 제목만 약 92자 기준으로 정리
- 카테고리별로 한 분야가 과도하게 몰리지 않도록 제한

## 기대 결과

```text
1. 석유 최고가격 L당 150원 인하…주유소 기름값 부담 완화
2. 코스피 급락에 서킷브레이커 발동…8400선 후퇴
3. 홈플러스 직원·협력사·점주, 국민신문고 통해 정부 지원 요청
```
