# v1.33 간밤의 핫이슈 근거 링크 + SNS 트렌드 강화

## 목적

간밤의 핫이슈 리포트에 신뢰 가능한 근거 링크를 더 많이 포함하고, SNS별 트렌드 후보를 추가합니다.

## 변경 내용

```text
1. 미국뉴스 주요 이슈별 근거 기사 링크 추가
2. 미증시 주요 지표별 Yahoo Finance 링크 추가
3. 국내뉴스 아침 이슈별 근거 기사 링크 추가
4. 간밤 인기 키워드/트렌드별 관련 기사 링크 추가
5. 실시간 급상승 키워드별 관련 기사 링크 추가
6. SNS별 트렌드 섹션 추가
7. 텔레그램 텍스트 리포트 자동 분할 전송
8. 카드뉴스에 SNS별 트렌드 페이지 추가
```

## SNS 트렌드 수집 방식

공식 SNS API 없이 안정적으로 운영하기 위해 v1.33은 우선 뉴스/검색 기반 보조 수집 방식을 사용합니다.

```text
YouTube: 유튜브 인기 영상 한국
Instagram/Reels: 인스타그램 릴스 트렌드 한국
TikTok: 틱톡 트렌드 한국
X/Threads: X 트렌드 한국 / Threads 인기 주제 한국
```

이 방식은 실제 플랫폼 내부 랭킹과 100% 같지는 않습니다.  
향후 YouTube Data API, TikTok/Instagram/X 공식 API 또는 제3자 트렌드 API를 붙이면 정확도를 더 높일 수 있습니다.

## 신규 환경변수

```yaml
OVERNIGHT_ENABLE_SNS_TRENDS: "true"
OVERNIGHT_SNS_TREND_QUERIES: "YouTube:유튜브 인기 영상 한국|Instagram/Reels:인스타그램 릴스 트렌드 한국|TikTok:틱톡 트렌드 한국|X/Threads:X 트렌드 한국|X/Threads:스레드 Threads 인기 주제 한국"
```

```text
간밤의 핫이슈 텍스트 리포트 (1/2)
간밤의 핫이슈 텍스트 리포트 (2/2)
```
