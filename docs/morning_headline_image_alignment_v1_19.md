# v1.19 헤드라인 뉴스 이미지-텍스트 정합성 개선

## 문제

헤드라인 뉴스 이미지는 생성되지만 카드 안의 bullet 내용이 실제 기사 제목과 잘 맞지 않았습니다.

예:
- 제목은 경제/산업 기사인데, bullet은 일반적/추상적 문장
- `세부 내용 확인 필요`, `분야 주요 이슈` 같은 placeholder 문구 반복
- 텍스트 메시지와 이미지 카드가 서로 다른 품질의 데이터를 사용

## 원인

1. 이미지 렌더링이 `headline_text`보다 `short_title`/기본 summary를 우선 사용
2. Gemini 실패 시 fallback summary가 너무 일반적
3. 이미지와 텍스트가 동일 데이터 구조를 공유하지 않음

## 수정

- 이미지 카드 제목은 `headline_text`를 우선 사용
- 카드 bullet은 기사 내용 기반 `summaries`만 사용
- 금지 문구:
  - 세부 내용 확인 필요
  - 분야 주요 이슈
  - 후속 기사 확인 필요
- fallback도 `title + description` 기반으로 생성
- 실행 로그에 briefs preview를 출력해 실제 생성 데이터 검수 가능

## 기대 결과

이미지 카드와 텍스트 메시지가 같은 핵심 문장을 공유하고,
bullet도 실제 기사 내용에 맞게 출력됩니다.
