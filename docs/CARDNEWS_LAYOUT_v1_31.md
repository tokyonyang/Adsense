# v1.31 카드뉴스 레이아웃 최적화

## 목적

v1.31은 사용자가 선택한 새 디자인 방향을 실제 자동화 템플릿에 반영합니다.

## 핵심 변경

```text
1. 커버 페이지를 프리미엄 경제 브리핑 스타일로 변경
2. 본문 페이지를 카드형 정보 구조로 재설계
3. 차트 / 인사이트 / 출처 / 키워드 영역을 독립 패널로 분리
4. footer 브랜드를 gooddaynews.store로 고정
5. 기존 "비티의 인사이트 노트" 문구 제거
6. workflow 파일 4개를 패치에 강제 포함
```

## 카드 구조

### 커버

```text
상단: 아침 브리핑 · 날짜
대제목: 오늘의 헤드라인 뉴스
설명: 경제·금융, 증시, 산업 이슈 중심
오늘의 흐름 요약 패널
gooddaynews.store footer
```

### 본문

```text
상단: 아침 브리핑 · 날짜 / 카테고리 / 페이지 번호
대제목: 주요 이슈 제목
태그: 기준 키워드
유사 기사 3건 핵심 요약
관련 지표 차트
한줄 인사이트
관련 키워드 / 출처
gooddaynews.store footer
```
