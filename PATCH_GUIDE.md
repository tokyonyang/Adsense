# AdSense SEO 운영 대시보드 v1.6 반응형 레이아웃 패치

## 목표

현재 대시보드가 PC에서는 넓게 보이고, 모바일에서는 카드/테이블/버튼이 깨지지 않도록 반응형 레이아웃을 강화했습니다.

## 주요 수정

- PC 화면 최대폭과 여백 정리
- 상단 버튼 줄바꿈 개선
- Supabase 상태 패널 레이아웃 정리
- 바로가기 카드 4열/2열/1열 반응형
- KPI 카드 6열/3열/2열/1열 반응형
- 태블릿 이하에서 사이드바를 상단 가로 메뉴로 전환
- 모바일에서 버튼을 한 줄 전체 폭으로 표시
- 테이블 가로 스크롤 안정화
- 콘텐츠 카드, 로그, 근거자료 카드 줄바꿈 개선
- 모바일에서 토스트 알림 위치/폭 최적화

## GitHub에 업로드할 파일

이번 패치는 설정 파일을 건드리지 않습니다.

아래 3개만 교체하세요.

```text
index.html
dashboard/index.html
vercel.json
```

`dashboard_config.js`는 기존 anon key가 들어간 상태로 그대로 두세요.
