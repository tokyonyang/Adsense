# Dashboard

`static_dashboard.html`은 현재까지 만든 대시보드 목업의 최신 버전입니다.

포함 기능:

- 카테고리형 키워드/뉴스 대시보드
- 회원제/사용자별 최적화
- 개인정보방침/이용약관/문의하기
- 콘텐츠 캘린더
- 근거자료 관리
- SEO 검수/애드센스 안전도
- 제목·썸네일 A/B
- SNS 트렌드
- 상품 소싱 후보

## 사용 방법

브라우저에서 직접 열어 UI를 확인합니다.

```text
dashboard/static_dashboard.html
```

실제 서비스화 단계에서는 이 파일을 Next.js 컴포넌트로 분리하는 것을 권장합니다.
